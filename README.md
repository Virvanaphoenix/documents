# documents
源码示例库


包分组说明：

1、algorithm			算法相关的类位于此包下</br>
2、demo				独立的小例子位于此包下</br>
3、designpattern		设计模式相关的例子位于此包下</br>
4、exam				与考试相关的例子位于此包下</br>
5、example			所有程序语言特性的例子位于此包下</br>
6、office			Office相关的操作位于此包下</br>
7、util				小工具包，提供其他类使用，为避免与公用冲突，不使用utils</br>
8、common			所有公用类位于此包下</br>
9、licence			授权包，建立软件授权机制</br>
10、jfreechart		jfreechart相关示例位于此包下</br>
